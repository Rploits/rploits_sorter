📌 Purpose:
This tool scans email addresses from a .txt file and detects:
 - Domain name
 - Email provider (Gmail, Outlook, Zoho, etc.)
 - Mail server (MX record)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
🚀 HOW TO USE:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1️⃣ Prepare Your Email List:
   - Open Notepad or Notepad++
   - Paste one email per line (no commas or extra characters)
   - Save the file as: emails.txt (or any .txt name)

2️⃣ Run the Tool:
   - Double-click `Rploits_Sorter.exe`
   - When prompted, type the full name of your .txt file (e.g., emails.txt)
   - Press ENTER to begin the scan

3️⃣ Review the Results:
   - The sorted output will be saved as:
     ✅ email_sorted_1.txt
     ✅ email_sorted_2.txt
     … (new file each run)

4️⃣ Want to Scan Again?
   - After it finishes, you’ll be asked if you want to scan a new file
   - Type Y (Yes) or N (No)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
⚠️ IMPORTANT NOTES:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

• Your file must be plain text (.txt)
• Do NOT use Excel or Word documents
• Each email must be on its own line (no commas or semicolons)
• Keep this tool in a simple folder like Desktop or Documents
• No internet = no results (tool checks mail servers live)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📂 Output File Columns:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Email                 → Full email address
  Domain                → The domain (e.g., gmail.com)
  Provider              → Gmail, Outlook, Zoho, etc.
  MX Records            → Mail server host (for tech users)
